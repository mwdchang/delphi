AnalysisGraph API¶

The central data structure for Delphi is the AnalysiGraph, which represents a directed graph that encodes information about causal quantitative relationships between concepts that we are interested in modeling.

The AnalysisGraph class currently inherits from the NetworkX DiGraph object, but the underlying implementation might be switched out for a faster one in the future.

Each node in the graph represents a concept - at the time of writing this (01/31/2019), this means an entry in an ontology of high-level concepts related to food security, like this one.

Each concept is associated with a set of indicators, which correspond to real-world, measurable quantities (and importantly, that Delphi has access to normalized data for.). In practice, this set of indicators is implemented as a dictionary, keyed by the name of the indicator. The values of the dictionary are objects of the class delphi.random_variables.Indicator.

The methods listed on this page constitute the public API for Delphi.

Constructors¶

AnalysisGraph.from_statements(sts[, …])

Construct an AnalysisGraph object from a list of INDRA statements.

AnalysisGraph.from_statements_file(file)

Construct an AnalysisGraph object from a pickle file containing a list of INDRA statements.

AnalysisGraph.from_text(text[, webservice])

Construct an AnalysisGraph object from text, using Eidos to perform machine reading.

AnalysisGraph.from_json_serialized_statements_list(…)

Construct an AnalysisGraph object from a list of JSON serialized INDRA statements.

AnalysisGraph.from_json_serialized_statements_file(file)

Construct an AnalysisGraph object from a file containing JSON-serialized INDRA statements.

AnalysisGraph.from_uncharted_json_file(file)

Construct an AnalysisGraph object from a file containing INDRA statements serialized exported by Uncharted’s CauseMos webapp.

AnalysisGraph.from_uncharted_json_serialized_dict(_dict)

Construct an AnalysisGraph object from a dict of INDRA statements exported by Uncharted’s CauseMos webapp.

Subgraphs¶

AnalysisGraph.get_subgraph_for_concept(concept)

Returns a new subgraph of the analysis graph for a single concept.

AnalysisGraph.get_subgraph_for_concept_pair(…)

Get subgraph comprised of simple paths between the source and the target.

AnalysisGraph.get_subgraph_for_concept_pairs(…)

Get subgraph comprised of simple paths between the source and the target.

Inspection¶

Methods for detailed inspection of the AnalysisGraph.

inspect_edge(G, source, target)

‘Drill down’ into an edge in the analysis graph and inspect its provenance.

Manipulation¶

Methods to edit the AnalysisGraph.

AnalysisGraph.merge_nodes(n1, n2[, …])

Merge node n1 into node n2, with the option to specify relative polarity.

AnalysisGraph.delete_node(node)

Removes a node if it is in the graph.

AnalysisGraph.delete_edge(source, target)

Removes an edge if it is in the graph.

AnalysisGraph.delete_nodes(nodes)

Iterate over a set of nodes and remove the ones that are present in the graph.

AnalysisGraph.delete_edges(edges)

Iterate over a set of edges and remove the ones that are present in the graph.

AnalysisGraph.prune([cutoff])

Prunes the CAG by removing redundant paths.

Parameterization¶

AnalysisGraph.map_concepts_to_indicators([…])

Map each concept node in the AnalysisGraph instance to one or more tangible quantities, known as ‘indicators’.

AnalysisGraph.parameterize([country, state, …])

Parameterize the analysis graph.

Sampling and Inference¶

AnalysisGraph.sample_from_prior()

Sample elements of the stochastic transition matrix from the prior distribution, based on gradable adjectives.

AnalysisGraph.infer_transition_matrix_coefficient_from_data(…)

Infer the distribution of a particular transition matrix coefficient from data.

Export¶

AnalysisGraph.to_dict()

Export the CAG to a dict that can be serialized to JSON.

AnalysisGraph.to_agraph([indicators, …])

Exports the CAG as a pygraphviz AGraph for visualization.
